JSON FORMAT FOR API CALL

(GET) assyrian-bible-api/books/ot/genesis/1
(Gets genesis chapter 1)

(GET) assyrian-bible-api/books
(Gets all books in old and new testament)

(GET) assyrian-bible-api/books/nt
(Gets all books in the new testament)

(GET) assyrian-bible-api/books/ot
(Gets all books in the old testament)
